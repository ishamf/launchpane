# Launchpane

Simple GUI command runner.

- Run short or long-running commands.
- Keep track of the log output.

## Requirements

- Node.js
- pnpm
- Rust installation, as in [Tauri prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites).

## Development

```sh
pnpm install
pnpm dev
```

## Building

```sh
pnpm build
```
